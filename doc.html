Postup instalace

Např. vyžadované prostředky na serveru, datové zdroje, inicializace databáze, apod. Stačí v několika stručných bodech.

Inštalácia pomocou docker-compose

docker-compose up spustí PostgreSQL databázu, aplikuje DB schému a migrácie, naplní DB dátami a spustí Django server na localhost:8000

Manuálna inštalácia

  1. Spustenie PostgreSQL DB na porte 5432, napríklad pomocou docker pull postgres && docker run --name postgres --publish 127.0.0.1:5432:5432 -d postgres
  2. Nainštalocanie dependencií
    • python3 -m venv .venv
    • source .venv/bin/activate
    • pip install -r requirements.txt
  3. cd mysite
  4. Aplikovanie migrácii a vytvorenie DB schémy python manage.py migrate
  5. Naplnenie DB python populate.py
  6. Spustenie serveru na localhost:8000 python manage.py runserver localhost:8000

Vyžadované knihovny

Django=2.0.2 psycopg2-binary django-extensions django-tables2 django-filter django-bootstrap-form
